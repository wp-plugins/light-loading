== Description ==

Adds a customizable loading screen to your blog whenever a page loads.

== Installation ==


1. Upload `light-loading` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I change the loading image? =

Goto admin->Appearance->Light Loader->play with the settings
